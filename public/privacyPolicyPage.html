Política de Privacidade

Última atualização: 29/04/2025

A sua privacidade é uma prioridade para nós. Esta Política de Privacidade explica de forma clara como coletamos, usamos, armazenamos e protegemos suas informações quando você utiliza nossa plataforma.

1. Informações Coletadas

Coletamos os seguintes tipos de dados por meio das integrações com APIs de terceiros:

  • Dados de Cadastro: Nome, e-mail e dados de autenticação via OAuth.
  • Credenciais de Plataformas: Tokens de acesso, identificadores de páginas (Facebook, Instagram) e credenciais de conta do Google Analytics.
  • Métricas Analíticas: Dados como alcance, impressões, cliques, sessões, visualizações de página, entre outros.

Esses dados são obtidos através das APIs oficiais das seguintes plataformas:

  • Meta (Facebook e Instagram)
  • Google Analytics (GA4)
  • Outras plataformas futuras (como TikTok, LinkedIn, etc.)

2. Uso das Informações

As informações coletadas são utilizadas para:

  • Autenticar usuários e permitir o acesso à plataforma.
  • Buscar e exibir métricas analíticas em dashboards personalizados.
  • Gerar relatórios e análises automatizadas para os usuários.
  • Melhorar a experiência e os recursos oferecidos na plataforma.

3. Compartilhamento de Dados

Não compartilhamos seus dados pessoais ou de plataformas integradas com terceiros, salvo quando:

  • Exigido por obrigações legais, judiciais ou regulatórias.
  • Necessário para prevenir fraudes, proteger a segurança da plataforma ou garantir direitos legais.

4. Armazenamento e Segurança

Adotamos medidas técnicas, administrativas e organizacionais para proteger seus dados:

  • Tokens de acesso e informações sensíveis são criptografados e armazenados com segurança em bancos de dados protegidos.
  • Acesso restrito apenas a sistemas e pessoas autorizadas.
  • Monitoramento constante para detectar e prevenir acessos não autorizados.

5. OAuth e Permissões de API

Utilizamos OAuth 2.0 para autenticação segura com:

  • Facebook / Instagram (via Meta API)
  • Google Analytics (via Google Analytics Data API v1)

Solicitamos apenas as permissões mínimas necessárias para o funcionamento da plataforma. O acesso pode ser revogado a qualquer momento pelo usuário.

6. Uso de Dados para Modelos de IA/ML

Nosso sistema NÃO utiliza os dados obtidos das APIs do Google Workspace, como a Google Analytics Data API, para desenvolver, treinar ou aprimorar modelos generalizados de Inteligência Artificial ou Machine Learning.

O uso dos dados é restrito exclusivamente para fornecer análises específicas e personalizadas para cada usuário, sem uso para fins de treinamento externo.

7. Direitos do Usuário

Você pode a qualquer momento:

  • Acessar os dados que temos sobre você.
  • Solicitar correções ou exclusão de informações.
  • Revogar permissões de acesso concedidas anteriormente.

Para isso, entre em contato pelo nosso canal de suporte oficial.

8. Retenção de Dados

Os dados são armazenados pelo tempo necessário para fornecer nossos serviços, ou por prazos maiores se exigidos por lei. Você pode solicitar a exclusão a qualquer momento, conforme seus direitos descritos acima.

9. Atualizações da Política

Esta política pode ser atualizada periodicamente para refletir mudanças legais, operacionais ou de segurança. A versão mais atualizada estará sempre disponível nesta página.
